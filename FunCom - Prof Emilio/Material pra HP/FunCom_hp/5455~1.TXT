	Tal sinal é demodulado usando um detector coerente.

	O discriminador produz um sinal de saída cuja amplitude é proporcional a frequencia instantanea da portadora.

5.5.2 Efeito Limiar em FM

	Assumindo-se uma SNR alta observa-se que as componentes de sinal ruido na saida do discriminador FM são aditivas.
	Em geral esta condição não é correta.
	Para baixos valores de SNR as componentes de sinal e ruido apresentam-se tão interligadas que é impossivel distinguir o sinal desejado do ruido  (devido a nao linearidade do processo de demodulação).
	Existe um valor especifico para SNRc, abaixo do qual, este fenomeno ocorre e o receptor FM deixa de funcionar (efeito limiar).
	A literatura mostra que SNR de limiar é dada por:

	SNR_limiar =~ 20.(Beta+1)

	Esta expressãop permite calcular o maximo valor permitido de Beta de modo a assegura que o sistema trabalha acima do limiar.

5.5.3 Pré Ênfase e De-Ênfase em FM

	isto significa que para baixas frequencias o desempenho da demodulação FM é melhor do que para altas frequencias. 
	O metodo de pre(de)-enfase permite equalisar o desempenho ao longo de toda faixa de frequencia, conforme mostrado abaixo:

	     _______      _______           ______      _______
	    |filtro |    |       |   +     |      |    |filtro |   m(t)+n(t)
	--->|pre    |--->|Tx FM  |--->O--->|Rx FM |--->|de     |--->
            |enfase |    |_______|    ^    |______|    |enfase |
                                      |
                                  n(t)|

	As componentes de alat ferquencia de m(t) são artificialmente enfatizadas antes da modulação, e antes que o ruido seja introduzido. Na saida do discriminadorrealiza-se uma operação inversa atenuando as componentes de alta frequencia e, assim, aumentando a SNR de saida do sistema.
	No sistema FM comercial utilizam-se filtros RC (1a ordem) passa-baixa para de-enfase e passa alta para pré enfase.

		H_de = 1/(1-j(f/fo))

		H_pre = 1+ j(f/fo)

	Como os efeitos dos filtros são cancelados, a potência de m(t) permanece inalterada e deve-se considerar somente o efeito de filtragem (FPB) sobre o ruido recebido.
