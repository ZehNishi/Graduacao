Instruções para uso:

Folha resposta:
	Modelo criado no draw.io, disponivel na pasta Template_Garito,
	como png e como modelo completo.

Provas:
	Todas provas foram salvas no caminho Provas desse diretório.

Relatório e apresentação:
	Relatório feito no overleaf, disponível no diretório Relatório,
	em pdf e .zip, e apresentação feita no canva, disponível em pdf no mesmo diretório.

Scripts:
	OMR.m:
		Análise de apenas uma folha resposta, com plots do passo
		a passo do código, mudar o caminho da imagem para a imagem
		a ser análisada.
	Calcula_Notas.m:
		Realiza a análise completa do projeto, requer no mínimo uma
		folha resposta com o nome Gabarito, com o tipo de prova 
		especifícado, demais provas devem ter o nome do responsável
		pela mesma e todas folhas respostas devem estar dentro do
		diretório Provas.
		Rodar o código retorna a nota de cada folha resposta de acordo
		com seu tipo de prova.
